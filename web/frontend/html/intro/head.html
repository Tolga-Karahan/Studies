What's in the head of HTML???

Content of HTML head section doesn't appear on pages. It contains information such as the page title, links to CSS, custom
favicons and other metadata. Browsers use the information in <head> section to render the page accordingly. <title>
element in the <head> is title of the overall HTML document. Sometimes <h1> tag in the body also referred as title of the
page. While <h1> is the title of the content that is visible on the browser, title in the head section is title of the all document.
Title contents are also used in the search results.

<meta> tag in head section is used to add metadata to the document. <meta charset="utf-8"> is common metadata
in HTML documents. It defines the document's character encoding. utf-8 is sufficient for any character from any language.
For example to render Japanese we need it: ご飯が熱い。.

Many metadata information has a name to indicate what type of metadata it is and a content. Some metadata is used by
search engines so defining them is a good practice for SEO. For example content of description metadata is shown in the
search results.

You can add references to custom icons in metadata to further enrich the design. Most common one is favicon which
appears in favorites or bookmarks lists. It might appear on tab as well depending on the browser:
<link rel="icon" href="favicon.ico" type="image/x-icon">

We can set the language of the page in head as well. In this way the page will be indexed better by search engines to
show it in the language specific results.
