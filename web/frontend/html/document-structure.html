Basics of HTML Document Structure

There are several HTML block elements to define areas of websites such as header, navigation menu, main content column.

Basic sections of a document
header
Usually a big strip across the top with a big heading, logo and a possible tagline.
navigation bar
Contains links to the sections of the site. Usually includes menu buttons, links, or tabs.
main content
Big area in the center that is usually unique for each page.
sidebar
Some additional info related to main content. For example if the main content is news then sidebar may include info related to author.
Sometimes behaves like a seconday navigation system.
footer
A strip across the bottom of the page. Usually contains fine print, copyright, or contact info.
