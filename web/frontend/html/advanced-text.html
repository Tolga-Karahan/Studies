There is also a third type of list in HTML which is called description lists. Description lists are used in cases in which a set of items and their descriptions are shown.
For example terms and definitions or Q/A. We can use multiple description per item.

Unordered list
A list consists of elements whose order doesn't matter.
Ordered list
A list consists of elements whose order does matter.
Definition list
A list consists of items and their descriptions as it is the case with this one.
Though it is not frequent as previous ones.

The HTML <blockquote> Element (or HTML Block Quotation Element) indicates that the enclosed text is an extended quotation.

There are also some tags which are useful for representing computer code:
<code>
For marking up computer code.
<pre>
For preserving spaces in HTML as is otherwise they are just ignored from the browser.
<var>
For specifically marking up variable names.
<kbd>
For marking up keyboard or any other type of input entered into computer.
<samp>
For marking up the output of a computer program.

There are multiple ways of writing dates, but computers might not be able to recognize each one. To indicate that it's a date we use <time> tag. By doing so we
can easily grab all dates in a page.
