Responsive Images

Responsive images are images that work on devices with different screen sizes, resolutions, and other features.

How to create responsive images?

Resolution switching: Different sizes

srcset and sizes attributes of <img> element can be used to render same image with different resolutions
according to the device. srcset attribute contains a set of images and their respective width. sizes attribute
defines a set of media conditions for example screen width to determine which image size would be best
and along with slot widths to determine the width of the slot the image will fill if the condition is true.
Browser go through conditions and gets the first matching by ignoring rest. One without a condition in the
end is the default size. If images in srcset doesn't exactly match with the chosen size from sizes attribute
than the first bigger one is chosen.

Resolution switching: Same size, different resolutions

If multiple resolutions are supported then browser can be allowed to choose an appropriate resolution
from srcset attribute. In below example original image which is 320pixels wide is supplied along with 1.5x
and 2x resolution ones. Image width specified as 320px via CSS. The browser works out to pick an appropriate
one. If for example device has standard/low resolution with one device pixel representing one CSS pixel then
the original image is chosen. If device has a high resolution of two device pixels per CSS pixel then 2x one is
chosen.

Art direction

Art direction problem is changing the image displayed to suit different display sizes. For example a person
in the middle of a big landscape image would be hardly visible in a mobile browser. It would be better to
show a smaller image that focuses more on the person. This kind of solution can be implemented via
<picture> element. <picture> element also has <source> elements inside that provides different sources
for the browser to choose one from. <source> element also has a media attribute which defines a media
condition and a srcset to pick from one or multiple images. Even sizes attribute also can be added for
different resolutions. Inside <picture> element, an <img> element also should be added as a default
image in case no condition is true and for the browsers that doesn't support <picture> element.
