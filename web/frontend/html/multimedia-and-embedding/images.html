Images in HTML

<img> tag

We use <img> tag to put an image on a web page. It's a void element meaning that it can't have any child element and an end tag.
It requires at least two attributes to be useful: src and alt. Filenames are also considered by SEO, so it is wise to give meaningful names.
There are also some other useful attributes such as width, height and title. width and height as their names imply specify the width
and height of the image in case it is not loaded to render an appropriate box. title attribute provides information when user hover
cursor over the image. Due to it requires a mouse to hover and accessibility problems with screen readers it is better practice to
give information in main text.
