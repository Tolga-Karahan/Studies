There are two categories of elements in HTML: block-level elements and inline elements.
Block level elements are visible blocks on a page such as headings, paragraphs, lists,
navigation menus, footers etc. They appear on a new line and any content that follows them
also appears on a new line. A block-level element can't be nested in a inline element, but
they can be nested in another block-level element. On the contrary, inline elements are inside
block elements. An inline element won't cause a new line to appear in the document. Some
examples are a, em, strong. These two categories are dropped in HTML 5 definitions, because
they cause misunderstanding with CSS block and inline boxes.
